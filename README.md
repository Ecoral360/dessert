# Dessert 🍰 : The deserializer and serializer for C3


## Features

### Serialization
- [X] Serialize to JSON
- [X] Recursive struct serialization (if an inner member of a struct is a pointer to the same struct)
- [X] Serialize Maybe fields
- [X] Serialize slice fields
- [X] Serialize struct fields
- [ ] Serialize enum fields
- [ ] Serialize union fields

- [X] Skip field
- [X] Rename field
